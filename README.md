Aplicativo para el registro dee cuotas de mantenimiento de una Urbanizacion, desarrollado en:
-	Frontend (Capas de Presentación): Desarrolla en Vue.js (JavaScript) para el navegador.
-	Backend (Capas de Lógica/Datos): Desarrolla el Microservicio de Transacciones en Python (usando Flask) para asegurar la lógica y la integridad de los datos financieros.
